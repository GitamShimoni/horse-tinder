**Button** — Trotr's primary action element; use `primary` (neon gradient) for the main CTA, `gold`/`emerald` inside the marketplace, `secondary`/`ghost` for lower emphasis.

```jsx
<Button variant="primary" iconLeft="zap">Go Premium</Button>
<Button variant="secondary" size="sm">Skip</Button>
<Button variant="gold" iconLeft="crown">Buy now</Button>
```

Variants: primary · secondary · ghost · gold · emerald · danger. Sizes: sm · md · lg. Props: `iconLeft`, `iconRight` (Icon names), `block`, `disabled`. Hover lifts + intensifies the glow ring.
