**Badge** — tiny uppercase status pill. `gold` = premium, `emerald` = verified, `neon` = hot/new, `super` = super-like, `neutral` = metadata.

```jsx
<Badge tone="gold" icon="crown">Neigh-Premium</Badge>
<Badge tone="emerald" icon="shieldCheck">Verified Stud</Badge>
```

Solid tones (neon/gold/emerald) carry a shadow; soft tones (super/neutral) are translucent.
